Developer's plugin for support of WordPress products.

== Description ==

You can to give support for buyers of your themes and plugins of more convenient way with Ticketrilla!

See how Ticketrilla works on demo sites.
We recommend this order:
1. Go to  ["Ticketrilla: Client" demo](http://client-demo.ticketrilla.com/wp-admin)
2. Create a ticket
3. Go to ["Ticketrilla: Server" demo](http://server-demo.ticketrilla.com/wp-admin)
4. Find a ticket and answer it. Look what happened. Convenient, isn't it?)

For pre-sales questions please visit [our site](https://ticketrilla.com/).

<h3>How it works?</h3>
[youtube https://youtu.be/qAK3dbnf8fY].
Ticketrilla works very simply - directly connects the  developer's and client's admin panels through plugins.
The developer installs the "Ticketrilla: Server" plugin (this item). After that clients can connect to it through ["Ticketrilla: Client"](https://wordpress.org/plugins/ticketrilla/) and receive support.

<h3>Main features</h3>
<ul>
<li>Free. Basic functionality without any payments. Absolutely</li>
<li>Built-in licensing system</li>
<li>Total control over the use of licenses at the addresses of clients' sites and servers</li>
<li>Unlimited agents and tickets</li>
<li>Full independence from third-side services and servers</li>
<li>Full security of file transfering, including the executable code</li>
<li>Duplication of tickets on client's server. Client can view tickets even when the your server is disconnected</li>
<li>Additional functionality from many addons</li>
</ul>

* Ticketrilla does not provide support and is not responsible for its provision. It only provides the opportunity to receive it without leaving the site.
** In free version official addons is not worked and you can't support. If you want you can to buy full version on [our site](https://ticketrilla.com).
